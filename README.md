## `~/engineering-principles`

<div align="center">

| Principle | In Practice |
|:---:|:---|
| 🏛️ **Architecture First** | Design the system before writing a single line of code |
| ♻️ **Automate Everything** | If you do it twice, write a script. If three times, build a service |
| 📐 **Clean > Clever** | Readable code is maintainable code. Clever code is a liability |
| 📊 **Measure, Don't Guess** | Benchmark before optimizing. Logs before debugging |
| 🔄 **Iterate in Production** | Ship small, validate fast, improve continuously |
